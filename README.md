## GitHub Pages Setup

1. Push this folder to a GitHub repository.
2. Open repository **Settings**.
3. Go to **Pages**.
4. Select:
   - Source: Deploy from a branch
   - Branch: main
   - Folder: /root
5. Save.

The website will be available at:

```text
https://hydraulic-pipo.github.io/WaRee11_Website/
```
